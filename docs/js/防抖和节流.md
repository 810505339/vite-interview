# 防抖和节流

  ## 防抖
  ***为什么需要防抖?***
当函数绑定一些持续触发的事件如：resize、scroll、mousemove ，键盘输入，多次快速click等等，
如果每次触发都要执行一次函数，会带来性能下降，资源请求太频繁等问题

非立即执行版本:
```ts
function debounce(fn,wait){
  let timer=null
  return (...args)=>{
     const context = this
      if(timer)
      clearTimeout(timer)
      timer=setTimeout(()=>{
        fn.apply(context,args)
      },wait)
  }
}
```
